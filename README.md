# Sayı Tahmin Oyunu

Flutter ile yazdığım sayı tahmin uygulamasıdır.
Bu uygulamada, kullanıcıdan 0 ve 100 dahil olmak üzere, 10'un katlarında bir sayı tahmini yapması istenir. 
Kullanıcı, seçilmiş sayıyı bulmaya çalışırken 4 adet tahmin hakkına sahiptir. Tahmin hakları bitene kadar, doğru sayıyı bulmak için bilgi ekranında yer alan ipuçlarından faydalanabilir.
